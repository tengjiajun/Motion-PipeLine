# Tools

Small shared utilities for path handling, manifest validation, metrics, and visualization.

| File | Purpose |
|---|---|
| `smpl_npy_to_gif.py` | Render AlphaPose/SMPL npy as skeleton GIF. |
| `fromw1_623_npy_to_gif.py` | Render FRoM-W1 623 feature npy as skeleton GIF. |
| `fromw1_623_to_pkl_batch.py` | Batch retarget FRoM-W1 623 feature npy into H1 pkl motions. |
| `robojudo_h1_pkl_batch_export.py` | Export RoboJuDo H1 pkl result visualizations. |
